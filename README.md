# Практическая работа №1 — Безопасность в сфере электронных услуг и сервисов.

**Вариант 25:** Безопасность в сфере электронных услуг и сервисов

## Описание
Консольное приложение для мониторинга событий безопасности в электронных сервисах (интернет-банкинг, платежные системы, почтовые сервисы).

## Функции
- **Авторизация пользователей** (email + пароль)
- **Просмотр событий безопасности** (пользователь видит только свои, администратор — все)
- **Добавление и редактирование событий**
- **Блокировка пользователей** (только для администратора)

## Запуск
Для запуска приложения используйте:
```bash
python3 pr1.py
```
